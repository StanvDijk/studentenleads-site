# StudentenLeads — Landing Site

Public landing page for StudentenLeads — a Dutch B2B lead-generation service that delivers verified warm leads via authentic student outreach on LinkedIn.

## Live site

Live via GitHub Pages at:
`https://stanvdijk1.github.io/studentenleads-site/`
(activeer onder Settings → Pages → Source: main, folder: /root)

## Local preview

Open `index.html` directly in your browser:
```
open index.html
```

## Structure

```
studentenleads-site/
├── index.html       # Single-page landing site (self-contained HTML + CSS)
├── images/          # Static assets
│   └── stan.jpg     # Profile photo (to be added by Stan)
└── README.md        # This file
```

## Custom domain setup (later)

Once `studentenleads.nl` is registered:

1. Create a `CNAME` file in repo root with the domain name
2. Configure DNS: point domain to GitHub Pages
3. Enable HTTPS in Settings → Pages
